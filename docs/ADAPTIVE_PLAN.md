# План адаптивной верстки сайта LARO

## Общие принципы 

- **Mobile-first подход** - начинаем с мобильных устройств, затем расширяем для больших экранов
- **Breakpoints**: `sm` (640px), `md` (768px), `lg` (1024px), `xl` (1280px)
- **Минимальный размер кликабельных элементов**: 44×44px (стандарт для touch-устройств)
- **Оптимизация типографики** для мобильных устройств
- **Адаптивные отступы и spacing** между элементами

---

## Секция 1: Header / Navigation

### Текущие проблемы:
- Навигация в одну строку не помещается на мобильных устройствах
- Логотип и текст могут перекрываться
- Нет мобильного меню

### План адаптации:

#### 1.1 Мобильное меню (гамбургер)
- Скрыть десктоп-навигацию на `<md` breakpoint
- Добавить кнопку-гамбургер справа в header
- Реализовать выдвижное меню (slide-in или overlay)
- Добавить функционал закрытия по клику вне меню или на ссылку

#### 1.2 Логотип и брендинг
- Уменьшить размер логотипа на мобильных (`w-12 h-12` → `w-10 h-10` или `w-8 h-8`)
- Скрыть слоган "Fast. Certain. Everywhere." на `<md`
- Уменьшить размер названия "LARO" если необходимо

#### 1.3 Sticky header
- Сохранить sticky поведение на всех устройствах
- Уменьшить высоту на мобильных (`h-[70px]` → `h-[60px]`)

#### 1.4 Backdrop blur
- Упростить/уменьшить blur на мобильных для производительности

## Секция 2: Hero

### Текущие проблемы:
- `grid-cols-2` не работает на мобильных
- Заголовок `text-7xl` слишком большой
- Кнопки в ряд могут не помещаться
- Контакты и соцсети в ряд переполняются

### План адаптации:

#### 2.1 Сетка и расположение
- Изменить `grid-cols-2` → `grid-cols-1` на `<lg`
- Поменять порядок: текст сверху, изображение снизу на мобильных
- Убрать `gap-16` → `gap-8` на мобильных

#### 2.2 Типографика
- Заголовок: `text-7xl` → `text-4xl` на мобильных, `text-5xl` на `sm`, `text-6xl` на `md`
- Описание: `text-[21px]` → `text-base` на мобильных, `text-lg` на `sm`
- Уменьшить `mb-6` → `mb-4` на мобильных

#### 2.3 Кнопки CTA
- Вертикальная компоновка на `<md`: `flex-col`
- Полная ширина на мобильных: `w-full`
- Уменьшить padding: `px-[32px]` → `px-6` на мобильных

#### 2.4 Контакты (телефон/email)
- Вертикальная компоновка на `<md`
- Уменьшить размер иконок: `text-2xl` → `text-xl`
- Обернуть в `flex-col gap-2`

#### 2.5 Соцсети
- Оставить горизонтально, но уменьшить размер: `w-10 h-10` → `w-9 h-9` на мобильных
- Уменьшить gap: `gap-3` → `gap-2`

#### 2.6 Изображение
- Убрать 3D transform на мобильных
- Уменьшить padding секции: `py-20` → `py-12` на мобильных
- `min-h-[90vh]` → `min-h-[70vh]` на мобильных
